# 坦克大战游戏调试指南

## 黑屏问题排查

### 问题现象
- 创建房间后，屏幕上只有黑色背景，看不到坦克

### 可能原因及解决方案

#### 1. 玩家数据未正确初始化
**问题**: 玩家位置数据未正确设置
**解决方案**: 已在代码中为玩家设置默认位置

#### 2. 数据库查询延迟
**问题**: 从数据库获取玩家数据有一定延迟
**解决方案**: 
- 确保数据库表已正确创建
- 检查网络连接是否稳定
- 重新加载页面

#### 3. 实时订阅问题
**问题**: Supabase 实时功能未正确订阅
**解决方案**:
- 检查 Supabase 项目的实时功能是否已启用
- 确认 RLS 策略配置正确

### 验证步骤

1. **确认数据库表存在**:
   - 在 Supabase 仪表板的 Table Editor 中确认以下表已存在：
     - game_rooms
     - game_players
     - game_bullets
     - game_events

2. **检查玩家数据**:
   - 在 game_players 表中查看是否有玩家记录
   - 确认 tank_x 和 tank_y 字段有有效数值

3. **浏览器开发者工具**:
   - 按 F12 打开开发者工具
   - 查看 Console 标签页是否有错误信息
   - 查看 Network 标签页确认 API 请求是否成功

4. **重启应用**:
   - 停止当前运行的应用 (Ctrl+C)
   - 重新运行 `npm run dev`

### 期望行为

- 创建房间后，玩家1的坦克应出现在左下角
- 加入房间后，玩家2的坦克应出现在右下角
- 坦克应显示健康条
- 方向键应能控制坦克移动
- 空格键应能发射子弹

### 常见错误信息
- "Could not find the table..." - 数据库表未创建
- "auth.role() = 'authenticated'" - 用户未登录
- "Connection refused" - 网络连接问题

如果问题持续存在，请检查浏览器控制台中的具体错误信息。
