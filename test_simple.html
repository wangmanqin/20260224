步骤 1: 运行 RLS 修复脚本

复制以下代码到 Supabase SQL 编辑器并运行：

-- 最简单的方法：完全禁用 RLS ALTER TABLE storage.objects DISABLE ROW LEVEL SECURITY; -- 验证 RLS 已禁用 SELECT schemaname, tablename, rowsecurity FROM pg_tables WHERE schemaname = 'storage' AND tablename = 'objects';

运行后应该看到 rowsecurity = f 表示 RLS 已禁用。
