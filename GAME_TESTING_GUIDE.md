- [ ] 子弹发射事件正确广播

## 测试步骤

### 单人测试（基础功能）
1. 访问 `http://localhost:3000/game`
2. 点击"创建房间"
3. 验证：
   - 房间成功创建
   - 显示正确的房间ID
   - 可以使用方向键移动坦克
   - 可以按空格键发射子弹
   - 坦克和子弹正确渲染

### 双人测试（完整功能）
1. 用户A访问 `http://localhost:3000/game` 并创建房间
2. 记录下房间ID
3. 用户B访问 `http://localhost:3000/game` 并使用房间ID加入
4. 验证：
   - 两个玩家都能看到对方的坦克
   - 移动操作实时同步到对方屏幕
   - 子弹发射实时同步到对方屏幕
   - 碰撞检测正确工作
   - 游戏结束条件正确触发

## 常见问题排查

### 问题1：无法创建房间
- 检查是否已登录
- 检查数据库连接
- 检查 `game_rooms` 表是否存在

### 问题2：实时功能不工作
- 检查 Supabase 实时功能是否启用
- 检查网络连接
- 查看浏览器控制台是否有错误

### 问题3：碰撞检测不准确
- 检查坐标转换是否正确
- 检查碰撞检测算法

## 性能优化建议

1. 减少不必要的数据库更新
2. 限制每秒更新频率
3. 优化 Canvas 渲染性能
4. 实现子弹和对象清理机制

## 扩展功能建议

1. 添加障碍物和地图
2. 实现不同类型的子弹或能力
3. 添加音效
4. 实现更复杂的 AI 对手
5. 添加游戏统计功能
